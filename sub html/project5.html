The example you see above is my first creation with my verlet physics engine. My projects goal was to understand how physics are replecated in code.

In the first code snipped i make the crane entity and give him the vertices. In my engine you can make new vertices by calling a entity and then the method .addDot(x, y); . Afther i gave the crane vertices i connect the vertices with the sticks. You use the method .addStick(and you use the vertex index in the array, and a second vertex index); .

In the second code snipped i show my entity class that i use and the methods it has. in this demo i only use addDot() addStick() and update(), to set the points and the sticks and unpdating them.
